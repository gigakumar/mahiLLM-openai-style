On‑device Assistant

Privacy‑first AI that automates personal productivity entirely on your device, with optional encrypted sync.

Product overview

  • Privacy‑first on‑device AI assistant for personal productivity
  • Automates scheduling, email drafts, summaries, and cross‑app workflows
  • Runs fully offline by default; optional encrypted sync
  • Tiered models for low‑power phones to desktops

Key features

  • Local RAG over an encrypted personal index
  • Task orchestrator converts LLM outputs into safe, auditable actions
  • Plugin system for Calendar, Gmail, Notion, WhatsApp Desktop
  • Personalization with local LoRA adapters and on‑device fine‑tune buffer
  • Progressive responses: instant summary then detailed follow‑up

Security & privacy

  • Data encrypted at rest; keys in device keystore
  • Hardware‑backed keys / Secure Enclave support
  • Signed plugin manifests in a sandbox
  • No network by default; external calls require explicit scope consent

Technical highlights

  • Rust core runtime with Kotlin/Swift bindings
  • Quantized GGML/ONNX with NNAPI/Metal acceleration
  • SQLite vector store with kNN + eviction policy
  • Signed, versioned model bundles; delta updates
  • CI with privacy/perf gates and plugin fuzzing

Business model

  • Freemium; Pro for larger models and integrations
  • Enterprise white‑label and fleet provisioning
  • Adapter marketplace for paid personalization packs
  • Value‑based upsells from usage patterns

Metrics & success

  • Task success rate and manual override frequency
  • Latency P50/P95 and memory/battery impact
  • Retention and Pro conversion
  • Privacy audit / compliance pass rate

Frequently asked questions

          
Answers to the most common questions about the console and platform.

        

        

          

            Can I connect my own model backend?
            
Yes. Host the Node server in this repo and set window.MAHI_API_BASE to point the UI at your endpoint.

          

          

            Is the demo streaming real?
            
On static hosting, we simulate streaming so the UX is identical. When connected to your backend, tokens stream over SSE.

          

          

            Do you keep my data?
            
In demo mode, nothing leaves your browser. When connected to your backend, data handling follows your server policies.

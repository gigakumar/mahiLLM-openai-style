Documentation

Everything you need to connect the console to your backend and deploy.

API

// POST /api/chat
// { messages: [{ role:"system|user|assistant", content: string }], model?: string, temperature?: number }
// SSE stream with events: data: { token?: string, done?: boolean }

Environment

  • MAHI_API_BASE (client): set to your server URL
  • PORT (server)
  • OPENAI_API_KEY, OPENAI_MODEL (server, optional)

Setup

  1. Deploy server.js to your host
  2. Set window.MAHI_API_BASE in index.html
  3. Reload the site and test streaming
